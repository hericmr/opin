Execute os seguintes comandos na raiz do projeto:

1. git add .
2. git commit -m "Atualiza página Alguns dados: adiciona imagens de header das escolas entre seções, ajusta cores (indigo em vez de teal), garante imagens únicas sem repetição, melhora performance do carregamento de imagens"
3. git push origin main
4. npm run build
5. npm run deploy

Ou execute o script: bash deploy.sh
